# 🌉 Kindora

> **Connecting Kindness with Action**

A production-ready **NGO Management System** built with **Python, Django, and PostgreSQL**.

---

## 📖 About

Kindora is a modern NGO Management System designed to help organizations manage:

- ❤️ Donors
- 🤝 Volunteers
- 📅 Events
- 📂 Projects
- 💰 Donations
- 📊 Reports
- 🧾 Invoices
- 🔔 Notifications

The goal is to provide a secure, scalable, and user-friendly platform for NGO operations.

---

## 🚀 Tech Stack

- Python 3.11+
- Django
- PostgreSQL
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Git & GitHub
